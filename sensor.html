The Lavender Sensor is designed specifically to simplify the collection of data as much as possible. The Sensor starts with a weather-proof housing, offering protection from the elements while recording data in the field. Next, the Sensor is solar powered with a 2500mAh backup batery system, and communicates wirelessly via bluetooth low energy. These features combine to create a seamlessly simple system that consists of only one required user action: switching on the power button.
